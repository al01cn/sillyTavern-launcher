## [0.1.6] - 2026-04-01

### 新增

- **鸣谢页面国际化**：设置页面新增「鸣谢」标签页，展示项目使用的技术栈与第三方库，描述文本支持中英文双语显示。

## [0.1.5] - 2026-03-30

### 新增

- **多种启动模式**：新增调试模式（开启 Node DevTools Inspector）、桌面程序模式（内嵌窗口，无需打开浏览器）、局域网模式（仅内网访问）、公网模式（全网可访问）共四种启动模式，可在设置页面切换。
- **局域网 / 公网链接面板**：局域网或公网模式启动酒馆后，首页按钮切换为「局域网链接」/「公网链接」，点击弹出专属面板，展示 IPv4 / IPv6 双二维码、地址与端口，一键复制或在浏览器打开。
- **公网可用性检测**：公网模式下，面板打开后自动通过 itdog TCPing（基于 Chrome Headless）检测 IPv4 / IPv6 节点超时率，并标注推荐协议。检测结果按服务启动次数缓存，同一次公网服务运行期间重复打开面板无需重新检测，点刷新按钮可强制重测。无 Chrome 时自动降级，提供 itdog 链接供手动查看。
- **对话历史管理**：资源页面新增「对话历史」标签页，按角色分组展示所有聊天记录，支持点击预览对话气泡、批量勾选删除。
- **GitHub 加速预加载与静态回退**：启动器启动时静默预加载 GitHub 加速地址列表；API 不可用时自动回退到内置备用加速地址，确保用户始终有可用选项。
- **GitHub 加速与 Node 版本联动**：Node ≥ 18.19.0 时使用 `--import` 拦截器注入加速；低版本自动改用 `git config --global url.xxx.insteadOf` 全局代理，服务停止时自动还原。
- **配置迁移**：在酒馆配置页面新增「迁移配置」按钮，支持一键迁移你之前使用的配置直接迁移到覆盖掉当前统一用户下配置。
- **资源迁移**：在资源页面新增「资源迁移」按钮，支持把你添加的酒馆里的角色卡、世界书等资源迁移与合并的形式，合并到当前统一用户下。

### 优化

- **扫描引擎升级**：采用 jwalk（并行）+ walkdir（顺序）双引擎策略，jwalk 5 秒内无结果自动切换 walkdir，彻底解决因权限问题导致扫描卡死的问题。
- **首次扫描仅执行一次**：首次启动完成自动扫描后不再重复触发，用户仍可手动发起扫描。
- **扫描路径实时日志**：扫描过程中可点击日志按钮实时查看当前扫描路径，方便排查异常。
- **扩展按需加载**：移除启动时预加载扩展列表，改为进入扩展页面时按需加载，加快启动速度。
- **桌面模式关闭体验**：桌面模式关闭子窗口时正确识别为主动停止，不再误报「异常退出」。
- **Git和Nodejs 环境切换功能**：提供自由切换到系统环境 或 内置环境。

### 修复

- **扫描大小写**：修复 `SillyTavern`（官方大写）无法被扫描识别的问题。
- **扫描误识别**：修复将安装目录下 `data/sillytavern/` 路径误识别为本地实例的问题。
- **手动添加酒馆**：修复 `package.json` 读取返回类型不匹配导致添加失败，以及 name 字段大小写校验不匹配的问题。
- **扩展修复路径**：修复第三方扩展自动修复和手动修复时路径错误的问题。

## [0.1.3] - 2026-03-27

### 新增

- **小白自动化一键安装**：针对全新小白用户新增全自动化“一键安装并启动”功能，串联 Node 环境安装、本体下载与一键启动的全流程。
- **本地实例极速扫描**：新增后台多线程扫描功能（基于 `jwalk` 和 `rayon`），静默全盘搜索本地已有的 SillyTavern 实例。并在设置中增加扫描时可使用的 CPU 核心数选项。
- **版本管理重构**：版本页面重组为“本地已安装”与“在线下载”双标签页，支持本地多路径实例的隔离管理和一键切换。
- **已有版本关联**：在版本管理页面新增“选择已有酒馆”按钮，可通过选择现成的 `package.json` 将其无缝连接到启动器中。
- **Git 环境管理**：在设置页面新增 Git 环境检测与跨平台一键安装支持。Windows 支持优先使用 Winget，并提供 PortableGit 备用下载方案。
- **国内网络优化自动配置**：新增区域检测，当系统地区为中国境内（含港澳台）时，初次启动将自动开启 GitHub 代理并自动将 npm 镜像源切换为淘宝源。
- **Git 集成**：提供轻量Git集成，方便让酒馆扩展支持自动更新。
- **Github 加速**：在酒馆里安装扩展时候请求的Github仓库时候，会自动替换成加速地址，如果你不需要可以在设置里关闭加速。

### 优化

- **本地版本切换优化**：不再依赖软链接创建目录，改为直接保存本地绝对路径。切换版本时自动更新程序的工作目录，避免因跨盘符或权限问题导致的启动失败。
- **启动器界面优化**：调整部分 UI 元素位置，优化启动器界面布局。
- **酒馆设置优化**：在安装依赖后自动生成优化过的设置文件，设置 关闭了酒馆的欢迎页面、两边面板锁定打开等优化。

### 移除

- **Soul公益站**：由于Soul公益站是类脑社区作者搭建的，并没有授权公开到类脑之外的地方，所以如果需要请到类脑社区自行查找使用。
