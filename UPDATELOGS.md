# 更新日志

本文件记录 SillyTavern Launcher GUI 的所有版本更新内容。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。

---

## [0.1.4] - 2026-03-29

### 修复

- **手动添加酒馆**：修复1.0.3版本中，手动添加酒馆时，无法正确识别 `package.json` 文件的问题。


## [0.1.3] - 2026-03-27

### 新增

- **小白自动化一键安装**：针对全新小白用户新增全自动化“一键安装并启动”功能，串联 Node 环境安装、本体下载与一键启动的全流程。
- **本地实例极速扫描**：新增后台多线程扫描功能（基于 `jwalk` 和 `rayon`），静默全盘搜索本地已有的 SillyTavern 实例。并在设置中增加扫描时可使用的 CPU 核心数选项。
- **版本管理重构**：版本页面重组为“本地已安装”与“在线下载”双标签页，支持本地多路径实例的隔离管理和一键切换。
- **已有版本关联**：在版本管理页面新增“选择已有酒馆”按钮，可通过选择现成的 `package.json` 将其无缝连接到启动器中。
- **Git 环境管理**：在设置页面新增 Git 环境检测与跨平台一键安装支持。Windows 支持优先使用 Winget，并提供 PortableGit 备用下载方案。
- **国内网络优化自动配置**：新增区域检测，当系统地区为中国境内（含港澳台）时，初次启动将自动开启 GitHub 代理并自动将 npm 镜像源切换为淘宝源。
- **Git 集成**：提供轻量Git集成，方便让酒馆扩展支持自动更新。
- **Github 加速**：在酒馆里安装扩展时候请求的Github仓库时候，会自动替换成加速地址，如果你不需要可以在设置里关闭加速。

### 优化

- **本地版本切换优化**：不再依赖软链接创建目录，改为直接保存本地绝对路径。切换版本时自动更新程序的工作目录，避免因跨盘符或权限问题导致的启动失败。
- **启动器界面优化**：调整部分 UI 元素位置，优化启动器界面布局。
- **酒馆设置优化**：在安装依赖后自动生成优化过的设置文件，设置 关闭了酒馆的欢迎页面、两边面板锁定打开等优化。

### 移除

- **Soul公益站**：由于Soul公益站是类脑社区作者搭建的，并没有授权公开到类脑之外的地方，所以如果需要请到类脑社区自行查找使用。

## [0.1.2] - 2026-03-24

### 新增

- Tavern 配置页面完整国际化支持（中英文）
- 内存缓存容量限制数字输入框（自动拼接 MB 单位）
- SSL 证书文件选择器（支持点击按钮选择文件）
- CORS 配置可视化列表 UI（支持动态添加/删除项）
- 所有数组类型配置改为列表形式 UI（IP 白名单、主机白名单、域名白名单等）
- 状态指示器国际化（同步中/就绪/正在加载配置）

### 优化

- 替换所有硬编码 SVG 图标为 @phosphor-icons/vue 组件
  - 删除按钮统一使用 PhX 图标
  - 添加按钮统一使用 PhPlus 图标
  - 文件选择按钮使用 PhFile 图标
- 折叠面板默认展开重要配置板块（SSL、CORS、代理与备份等）
- 缩略图尺寸输入框添加最小宽度限制，防止小屏幕重叠
- 代码量减少约 95%（SVG 迁移至图标组件）

### 修复

- 修复 IPv4/IPv6监听地址翻译键缺失问题
- 修复内存缓存容量需要手动输入 "mb" 后缀的问题
- 修复缩略图优化设置中输入框在小屏幕上重叠的问题
- 修复部分 UI 文本未国际化的问题

### 技术改进

- 实现 Vue 双向绑定自动处理单位后缀
- 使用 watch 监听实现配置实时同步
- 统一图标库使用，提升代码可维护性
- 完善翻译键管理，避免重复定义
