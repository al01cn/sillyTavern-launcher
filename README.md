# SillyTavern Launcher (酒馆启动器)

SillyTavern Launcher is a launcher application for SillyTavern, built with Tauri, Vue 3, and TypeScript.

## Project Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (or [Bun](https://bun.sh/))
- [Rust](https://www.rust-lang.org/)

### Install Dependencies

```bash
bun install
```

### Development

```bash
bun run tauri dev
```

### Build

```bash
bun run tauri build
```
